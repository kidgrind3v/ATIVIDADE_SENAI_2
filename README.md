# 🧮 Cálculo de Fatorial em Java

## 📖 Descrição do Projeto
Este programa solicita ao usuário um número inteiro e calcula o **fatorial** desse número utilizando uma estrutura de repetição `for`.  
O resultado é exibido diretamente no terminal.

O **fatorial** de um número é o produto de todos os inteiros positivos menores ou iguais a ele.  
Exemplo:  
`5! = 5 × 4 × 3 × 2 × 1 = 120`

---

## ⚙️ Funcionalidades
- 📥 Solicita um número inteiro ao usuário  
- 🔁 Calcula o fatorial do número informado  
- 📤 Exibe o resultado no terminal  

---

## 💻 Tecnologias utilizadas
- ☕ **Java**
- 📚 **Scanner** (para leitura de entrada do usuário)
- 🔂 Estrutura de repetição `for`
